# topfederalsites

A bot to turn data from [analytics.usa.gov](https://analytics.usa.gov) into a
Twitter bot.

## Installation

```
git clone https://github.com/stvnrlly/topfederalsites.git
cd topfederalsites
pip install -r requirements.txt
```
